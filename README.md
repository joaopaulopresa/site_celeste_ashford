# Celeste Ashford — Author Website

Static author website for Celeste Ashford (romantasy).

## Deploy on Cloudflare Pages

1. Push to GitHub
2. Connect repo to Cloudflare Pages
3. Build command: (none — static site)
4. Output directory: `/`

## Placeholders to Update

| Placeholder | Where | Notes |
|-------------|-------|-------|
| `AMAZON_LINK_MARROW_DECREE` | index.html | Amazon product link |
| `AMAZON_LINK_TITHE_THORNS` | index.html | Amazon product link |
| `AMAZON_LINK_HOLLOW_THRONE` | index.html | Amazon product link |
| `ARC_FORM_ID` | index.html | Formspree form ID for ARC requests |
| `NEWSLETTER_FORM_ID` | index.html | Formspree form ID for newsletter |
| `og-image.jpg` | assets/images/ | Open Graph social share image |
| `favicon.ico` | assets/images/ | Browser tab icon |
| `sample.pdf` / `sample.epub` | assets/files/ | Sample chapter files |

## Tech Stack

- HTML / CSS / JS (vanilla, no frameworks)
- Google Fonts (Cormorant Garamond + Raleway)
- Formspree (form handling)
- Cloudflare Pages (hosting)
